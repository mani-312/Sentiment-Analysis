# Sentiment-Analysis
A classification model that finds the sentiment of the movie review provided by user.
# Dataset
IMDB Movie Review dataset
# Preprocessing
- Converted each word to lowercase
- Removed Stopwords from each review
- Removed HTML tags, special characters from review.
- Converted each review into TfIdfVector

# Model
- Built a Logistic Regression model
- Interpreted the words that affect the sentiment of review.

# Accuracy
- Acheived F1 Score of 0.88 on test dataset.
